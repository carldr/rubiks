# Rubik's Solver

## Environment

- `brew install miniconda`
- `conda init zsh`

- `conda env create -f environment.yml`
- `conda activate rubiks`

## Development

- In VSCode, install ms-python, autopep8 (for formatting)
- Run `code .` after activating the Conda environment
- `F5` to run/debug

## Running

- `q` to quit the app
